Quick Start

Setup Steps

  1. Click Start below (or run npx tingly-box@latest in terminal)
  2. The Web UI URL is auto-detected — click Open Web UI to configure models
  3. Enter your Base URL below (default: http://localhost:12580/tingly/openai)
  4. Enter your API Token if required
  5. Click Save Configuration and reload VS Code window
  6. Open Language Models manager and enable model visibility

Need help? Visit vscode-tingly-box or tingly-box.

Custom Provider Setup

Steps

  1. Enter your provider's Base URL
  2. Enter your API Token if required
  3. Select the API style that matches your provider
  4. Click Test Connection to verify
  5. Click Save Configuration and reload VS Code window
