Tingly Box Configuration
Quick Start
  1. Click Start below, or run npx tingly-box@latest in terminal
  2. The Web UI URL is auto-detected — click Open Web UI to configure models
  3. Enter Base URL below (default: http://localhost:12580/tingly/openai)
  4. Click Save and reload the VS Code window
  5. Open Language Models manager and enable model visibility

Docs: vscode-tingly-box  ·  tingly-box

  4. Click Test Connection to verify
